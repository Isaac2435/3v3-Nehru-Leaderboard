# 3v3-Nehru-Leaderboard
A 3v3 debating tournament leaderboard in Excel (Macros)
Built for use in the Adelaide University Debating Society 3v3 competition - use for every round of the competition.
Tutorial within the file.

To use file, macros must be enabled. Macros are modules of code that edit the excel spreadsheet through data manipulation, formatting, etc.
To enable macros:
  1. Open the file and enable editing.
  2. If you have not already enabled macros, go to File > Options > Trust Centre > Trust Centre Settings > Macro Settings > Enable Macros
  3. Open file explorer and locate the file.
  4. Right click and open properties. Under 'General' find Security subheading and click "unblock".
For your security, check all updates to the file before use.
